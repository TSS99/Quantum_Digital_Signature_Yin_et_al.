# Quantum Digital Signature — Yin et al.

Implementation and analysis of the Quantum Digital Signature (QDS) scheme based on Yin et al.

## Contents

- `QDS_Yin_et_al.ipynb` — Main notebook with implementation, simulations, and results
- `attack_results.png` — Visualization of attack scenario results
- `performance_plots.png` — Performance benchmarks and plots

## Overview

This project implements the QDS protocol proposed by Yin et al., covering key generation, signing, verification, and security analysis including attack simulations.

## Requirements

Install dependencies via:
```bash
pip install -r requirements.txt
```
Or use the provided `.venv` (not tracked in version control).
